KnetMiner HumanDisease Release Notes

The final KnetMiner HumanDisease knowledge network contains:

    Datasets used for integration

    • Gene-Protein [Ensembl]
      • Ensembl BioMart
    • Disease [OMIM]
      • Alzheimer, Parkinson, Huntington diseases
    • Drug-Target Interactions [DrugBank]
    • Protein-Protein Interactions [IntAct]
      • Alzheimer Disease
      • Parkinson Disease
    • Protein-Publication [UniprotKB]
      • UniprotKB
    • Gene Ontology
      • GO
    • Gene-SNP-Phenotype [Ensembl]
      • Data was downloaded from Ensembl Biomart
